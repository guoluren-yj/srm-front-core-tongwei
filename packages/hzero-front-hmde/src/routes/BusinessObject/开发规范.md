### 开发规范（开发不使用mobx、dva等状态管理，使用dataSet做数据流转）

- 前端开发规范 - 目录结构只需要关注store的规范，这里的store是指dataSet

- 前端开发规范 - 国际化规范看同级文件多语言开发示例.md

- 前端开发规范 - ued规范以飞搭规范为准

- 一些通用组件不要引用带lowCode的

- 列表/表单内一些特殊的字段，需要用特定的renderer，如：是否必输（yesOrNoRenderer）、启用状态（enableRender）

```js
// 文档可查看 https://open.hand-china.com/document-center/doc/product/10137/10227?doc_id=32661&doc_code=32661， 里面还有一些其他的通用的方法/组件
// 引入
import { yesOrNoRender, enableRender } from 'hzero-front/lib/utils/renderer';
// 用法
{
  name: 'filterFlag',
  renderer: ({ value }) => yesOrNoRender(value),
},
{
  name: 'enabledFlag',
  width: 90,
  renderer: ({ value }) => enableRender(value),
},
``
